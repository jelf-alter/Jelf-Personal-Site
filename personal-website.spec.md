# Personal Website & Demo Platform Spec

## Overview
A personal website that serves as a resume supplement and platform for hosting small demo applications, showcasing development capabilities with polished frontend and robust testing.

## Requirements

### Core Features
- **Resume Integration**: Professional landing page with resume highlights
- **Demo Applications**: Showcase section for interactive demos
- **Testing Exposure**: Visible testing results and coverage metrics
- **Professional Polish**: Clean, modern UI/UX design

### Technical Requirements
- **Frontend**: Modern web framework (React/Vue/Svelte)
- **Backend**: API endpoints for demo applications
- **Testing**: Comprehensive test suite with public visibility
- **Hosting**: Deployable to common platforms (Vercel, Netlify, etc.)
- **Performance**: Fast loading, responsive design
